# Assignment 6 - Simple Web Server using Node.js

## Objective
Build a simple web server using Node.js http module and serve different HTML pages based on routes.

## Routes

- /home → Home Page
- /about → About Page
- /contact → Contact Page
- Invalid routes → Custom 404 Page

## Technologies Used

- Node.js
- HTTP Module
- HTML
- CSS

## How to Run

1. Clone the repository
2. Open terminal
3. Run:

   node server.js

4. Open browser:

   http://localhost:3000/home

## HTTP Status Codes

- 200 → Successful request
- 404 → Page not found
